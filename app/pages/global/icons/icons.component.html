Icons

Goose comes with a pre defined set of icons which can be easily used across any application.

Icon List

To create an icon, simply use the gs-icon component.

Browse the Full Icon List...

Basic Icon

To create an icon, simply use the iconName="bell" component input.

To create an icon, simply use the .g-icon and .g-icon-* class to any element.

¤

Primary Icon

You can have an icon display with the primary brand colour by configurating the iconStatus="primary" component input.

Notification Indicator

Icons can be used to display notifications by displaying a small red indicator at the top right hand corner of the icon.

To apply a notification indicator to an icon, simply add the iconStatus="alert" component input.

Extra classes Icon

You can have an icon that has multyple extra classes configurate it with iconClass="g-icon--primary g-icon--alert" component input.

Icon Sizes

If you require icons to be either bigger or smaller, config the component input iconSize.

Default size will be medium.
